Installation
------------

To use **SpaiNN**, first install it using pip:

.. code-block:: console

   (.venv) $ pip install spainn

Quick Start
----------------

To retrieve a list of ..., youi can use the :py:func:`lumache.get_random_ingredients` function: 

>>> import spainn as sp
>>> sp.get_random_ingredients()
['shells', 'gorgonzola', 'parsley']
